# AWMP Prerequisites

Before running the AWMP verification kernel, ensure you have the following tools installed and available in your PATH.

## Required Tools

| Tool | Purpose | Check Command |
|------|---------|---------------|
| `bash` | Shell execution | `bash --version` |
| `make` | Build/test orchestration | `make --version` |
| `rg` (ripgrep) | Text search | `rg --version` |
| `jq` | JSON parsing | `jq --version` |
| `sha256sum` | Hash verification | `sha256sum --version` |
| `git` | Repository operations | `git --version` |

## Installation

### Ubuntu/Debian
```bash
sudo apt-get update
sudo apt-get install -y bash make jq git
# ripgrep
sudo apt-get install -y ripgrep || cargo install ripgrep
```

### macOS
```bash
brew install bash make jq ripgrep git coreutils
```

## Troubleshooting

| Error | Cause | Fix |
|-------|-------|-----|
| `rg: command not found` | ripgrep not installed | `apt-get install ripgrep` or `brew install ripgrep` |
| `sha256sum: not found` | Missing coreutils | `brew install coreutils` (macOS) |
| `jq: parse error` | Invalid JSON input | Check job JSON syntax |
| `make: *** No rule to make target` | Wrong directory | Ensure you're in `demo/awmp-demo/` |

## For Verifier Development

If you're modifying the verifier script itself:
- Test in a clean directory (no cached artifacts)
- Remove `evidence/` between runs for deterministic behavior
- Check exit codes explicitly: `echo $?`
